Uso del sexy Flebox

Modelo de layout que requiere la relacion de un contenedor (llamados flex-items). De esa manera, el flex-container define un contexto respecto al cual los flex-items van a organizarse y calcular sus dimensiones.

	.flex-container
	  .flex-item
	  .flex-item
	  .flex-item
	    .flex-item--child

En el ejemplo anterior, flex-item--child no se considera un flex-item porque estos deben ser solo hijos directos de un flex-container.

Un flex-item podria ser a su vez un flex-container. Se define un flex-container con la propiedad sigueiente:

	display: flex;

Ejemplo item y container a la vez:

	.container {
	  display: flex;
	}

	.container > .item {
	  display: flex;
	}
